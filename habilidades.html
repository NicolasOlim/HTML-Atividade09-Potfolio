Incio

Aqui você verá algumas da coisa que eu consigo fazer e desenvolver. Coisas que eu aprendi com muita luta, porém, me ajudaram a me preparar para o futuro. São ferramentas que eu sei trabalhar ou sei como funcionam e posso auxiliar no seu uso.

Durante o Senai conheci o python. Aprendi muitas coisa sobre essa ferramenta e como usa-la. Fiz alguns projetos e tive varias aulas, tenho os codigos bem marcados na minha memoria, então não terei dificuldade para me lembra de como funciona. O python e uma ferramenta bem simples e prefiro usa-la do que usar o java.

O java e uma ferramenta muito mais complexa que o java, não gosto muito dela mas sei sua importancia para o area que estou cursando. Também fiz alguns projetos em java, sinceramente mais robusto do que em python, porém como uma certa dificuldade, porém não foi algo tão dificil. O java e bem complicado porém muito importante para meus planos.

O HTML é uma area que não gosto de mexer, frontend não e muito meu forte, e honestemente não sou muito criativo.Não gosto do CSS tenho muitas dificuldades para operar um CSS. Pretendo ir mais para o lado da backend como Python e Java. Mas sei o basico de HTML e caso precise vou poder ajudar com isso.

O javascript fooi bem legal de se trabalhar, as coisas complicaram quando ele se juntou com HTML, porém, não foi muito dificil, eu gosto de trabalhar com a backend então o javascript foi muito agradavél. Tenho conhecimento e posso trablhar com ele.

O Mysql melhe mais com o banco de dados, não foi dificil, o sql e bem sensivél a erros mas não tive dificuldades para opera-lo. Construir bancos e algo bem complicado porém não e muito dificil, estou alerta com a sensibilidade de seu sistema onde uma coisinha pode acabar com todo banco.

Vi muito pouco de raspberry mas vi um pouco do basico, com um pouco de mais de pratica vou estar muito melhor com o raspberry.
